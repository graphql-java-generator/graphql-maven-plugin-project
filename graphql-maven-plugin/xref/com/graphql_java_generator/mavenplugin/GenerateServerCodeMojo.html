22  /**
23   * <P>
24   * The <I>generateServerCode</I> Maven goal (and Gradle task) generates the java code for an almost ready to start
25   * GraphQL server. The developer has only to develop request to the data.
26   * </P>
27   * <P>
28   * The java code is generated from one or more GraphQL schemas. It allows to work in Java with graphQL, in a schema
29   * first approach. These items are generated:
30   * </P>
31   * <UL>
32   * <LI>the main method (in a jar project) or the main servlet (in a war project)</LI>
33   * <LI>All the GraphQL wiring, based on graphql-java-spring, itself being build on top of graphql-java</LI>
34   * <LI>All the POJOs, that contain the incoming request contents. The request response is written by the user code into
35   * these POJO, and the plugin take care of mapping them into the server response.</LI>
36   * <LI>An option allows to annotate the POJOs with the standard JPA annotations, to make it easy to link with a
37   * database. Please note that a</LI>
38   * <LI>All the interfaces for the {@link DataFetchersDelegate} (named providers in the graphql.org presentation) that
39   * the server needs to implement</LI>
40   * </UL>
41   * <P>
42   * The specific code that needs to be implemented is the access to the Data: your database, other APIs or web services,
43   * or any kind of storage you may have. This is done by implementing the interfaces for the {@link DataFetchersDelegate}
44   * into a Spring component, that is:
45   * </P>
46   * <UL>
47   * <LI>Create a class for each generated {@link DataFetchersDelegate} interface</LI>
48   * <LI>Make it implement the relevant {@link DataFetchersDelegate} interface</LI>
49   * <LI>Mark it with the {@link Component} annotation</LI>
50   * </UL>
51   * <P>
52   * And you're done! :)
53   * </P>
54   * <P>
55   * You'll find more info in the tutorials: take a look at the
56   * <A HREF="https://github.com/graphql-java-generator/GraphQL-Forum-Maven-Tutorial-server">Maven server tutorial</A> or
57   * the <A HREF="https://github.com/graphql-java-generator/GraphQL-Forum-Gradle-Tutorial-server">Gradle server
58   * tutorial</A>
59   * </P>
60   * 
61   * @author etienne-sf
62   */
